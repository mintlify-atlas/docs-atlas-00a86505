## SwitchboardError

Base error class for all Switchboard errors. All specific error types extend this class.

```typescript
class SwitchboardError extends Error {
  constructor(
    message: string,
    public readonly code: string,
    public readonly platform?: PlatformType
  )
}
```

<ResponseField name="message" type="string" required>
  Error message describing what went wrong
</ResponseField>

<ResponseField name="code" type="string" required>
  Error code for programmatic error handling
</ResponseField>

<ResponseField name="platform" type="PlatformType">
  Platform where the error occurred (if applicable)
</ResponseField>

<ResponseField name="name" type="string" required>
  Error name (always 'SwitchboardError' for base class)
</ResponseField>

## AdapterNotFoundError

Error thrown when no adapter is found for a platform.

```typescript
class AdapterNotFoundError extends SwitchboardError {
  constructor(platform: PlatformType)
}
```

**Error code**: `ADAPTER_NOT_FOUND`

**Example message**: `No adapter found for platform: discord. Did you import @aarekaz/switchboard-discord?`

<ResponseField name="platform" type="PlatformType" required>
  The platform that has no adapter registered
</ResponseField>

## ConnectionError

Error thrown when connection to a platform fails.

```typescript
class ConnectionError extends SwitchboardError {
  constructor(platform: PlatformType, cause: unknown)
}
```

**Error code**: `CONNECTION_ERROR`

**Example message**: `Failed to connect to discord: Invalid token`

<ResponseField name="platform" type="PlatformType" required>
  The platform that failed to connect
</ResponseField>

<ResponseField name="cause" type="unknown" required>
  The underlying error that caused the connection failure
</ResponseField>

## Error handling

All Switchboard errors include:
- A descriptive `message`
- A programmatic `code` for error type identification
- The `platform` where the error occurred (when applicable)
- The original `cause` error for debugging

```typescript Example error handling
try {
  await bot.sendMessage('channel-id', 'Hello!');
} catch (error) {
  if (error instanceof MessageSendError) {
    console.error(`Failed to send on ${error.platform}: ${error.message}`);
    console.error('Error code:', error.code);
    console.error('Original error:', error.cause);
  }
}
```
