## UnifiedEvent

Union type of all possible events that can be emitted by the bot.

```typescript
type UnifiedEvent =
  | MessageEvent
  | ReactionEvent
  | UserJoinedEvent
  | UserLeftEvent
  | ChannelCreatedEvent
  | ChannelDeletedEvent;
```

## MessageEvent

Event fired when a message is received.

```typescript
interface MessageEvent {
  type: 'message';
  message: UnifiedMessage;
}
```

<ResponseField name="type" type="'message'" required>
  Event type discriminator
</ResponseField>

<ResponseField name="message" type="UnifiedMessage" required>
  The received message object
</ResponseField>

## ReactionEvent

Event fired when a reaction is added or removed from a message.

```typescript
interface ReactionEvent {
  type: 'reaction';
  messageId: string;
  userId: string;
  emoji: string;
  action: 'added' | 'removed';
}
```

<ResponseField name="type" type="'reaction'" required>
  Event type discriminator
</ResponseField>

<ResponseField name="messageId" type="string" required>
  Message ID that was reacted to
</ResponseField>

<ResponseField name="userId" type="string" required>
  User ID who added/removed the reaction
</ResponseField>

<ResponseField name="emoji" type="string" required>
  Emoji string (e.g., '👍', 'tada')
</ResponseField>

<ResponseField name="action" type="'added' | 'removed'" required>
  Whether the reaction was added or removed
</ResponseField>
